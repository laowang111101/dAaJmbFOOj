# 教务管理系统简介 python795

## 项目概述

教务管理系统是一款基于Python的Flask框架开发的软件。该系统旨在为教育机构提供便捷的课程管理和学生信息管理服务。

## 技术栈

- 后端：Python 3.x
- Web框架：Flask
- 数据库：SQLite

## 功能模块

### 用户角色

- 老师
- 学生

### 主要功能

- 登陆与注册
- 开设课程
- 录入成绩
- 上传教学资源文件
- 课程管理
- 学生选课
- 查看个人信息

## 系统角色

- 老师：负责开设课程、录入成绩、上传教学资源。
- 学生：进行选课、查看个人信息和学习成绩。

## 运行环境

- Python 3.x
- Flask 2.x
- SQLite 3.x

## 部署步骤

1. 确保Python和Flask环境已正确安装。
2. 克隆或下载项目代码。
3. 安装项目依赖。
4. 配置数据库。
5. 运行应用。

## 目录结构

```
教务管理系统/
│
├── app/              # Flask应用目录
│   ├── __init__.py
│   ├── models.py     # 数据模型
│   ├── views.py      # 视图逻辑
│   └── templates/    # HTML模板目录
│
├── static/           # 静态文件目录
│
├── venv/             # 虚拟环境目录
│
└── requirements.txt  # 依赖文件
```

## 核心亮点

- 使用轻量级Web框架Flask，易于维护和扩展。
- 清晰的角色划分，保证数据的安全性和准确性。
- 提供全面的课程与学生信息管理功能，简化日常教务工作。
